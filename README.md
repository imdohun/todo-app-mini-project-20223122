# 풀스택 Todo 리스트 앱

React + Vite + Express + MongoDB Atlas + Vercel

## 프로젝트 구조
```
todo-app/
├── frontend/    # React + Vite
├── backend/     # Express API
└── vercel.json  # Vercel 배포 설정
```

## 기술 스택
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas
- **Deploy**: Vercel

## 기능
- Todo 추가
- Todo 목록 보기
- Todo 완료 체크
- Todo 삭제

## 실행 방법
### 백엔드
```bash
cd backend
npm run dev
```

### 프론트엔드
```bash
cd frontend
npm run dev
```
